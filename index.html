Projeto Engenharia de Software

O sistema que será criado fará parte de todo o processo de recrutamento e seleção, com o objetivo de informatizar e oferecer suporte as etapas do processo que na sua maioria são feitas manualmente. O sistema será desenvolvido para a plataforma Web/Mobile, com o objetivo de unir os recursos disponíveis em ambas as plataformas para uma melhor experiência, tanto para o candidato quanto para o usuário do sistema.
